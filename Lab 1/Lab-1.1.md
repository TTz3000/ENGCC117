## Computer Programming for Computer Engineer - Lab 1.1

# CODE
```c++
#include <stdio.h>

int main() {
    int N;
    scanf("%d", &N) ; // Input value

    for ( int i = 1; i <= N; i++)
    {
        printf("[%d] Hello world\n" , i);
    } //End loop
            
    return 0 ;
}
```
